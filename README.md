# angular_youtube_viewer

## first step

need install dev dependencies

```
   npm install .
```

## build static files

```
   npm run build
```   

## run test server
   
```   
   npm run serve
```   
   
all ready

just open http://127.0.0.1:8080/

## tests

```   
   npm test
```   

for test i use karma

After passing the test, coverage results will be in the appropriate folder (coverage)
